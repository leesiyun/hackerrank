# 없는 숫자 더하기

## 문제 설명

0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 `number`가 매개변수로 주어집니다.  
`numbers` 에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

<br />

## 제한 사항

- 1 ≤ `numbers`의 길이 ≤ 9
  - 0 ≤ `numbers`의 모든 원소 ≤ 9
  - `numbers`의 모든 원소는 서로 다릅니다.

<br />

## 입출력 예

|      number       | result |
| :---------------: | :----: |
| [1,2,3,4,6,7,8,0] |   14   |
|  [5,8,4,0,6,7,9]  |   6    |

<br />

## 입출역 예 설명

### 입출력 예 #1

- 5, 9가 `numbers`에 없으므로, 5 + 9 = 14를 return 해야 합니다.

### 입출력 예 #2

- 1, 2, 3이 `numbers`에 없으므로, 1 + 2 + 3 = return 해야합니다.

<br />

## solution.js

```JS
function solution(numbers) {
    // 0부터 9까지의 정수 배열
    let arr = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0];
    // 결과를 담을 변수
    let answer = 0;

    // 정수 배열의 값을 하나씩 가져온다.
    arr.map((value) => {
        // 주어진 배열에 가져온 값의 여부를 확인하고 false일 경우 결과 값에 더한다.
        if (!numbers.includes(value)) answer += value;
        // Array.prototype.map() expects a return value from arrow function
        return answer;
    });

    // 결과값을 반환한다.
    return answer;
}
```
