# Plus Minus

🔗 [Problem](https://www.hackerrank.com/challenges/plus-minus/problem?isFullScreen=true)

정수 배열이 주어지면 양수 , 음수 및 0 인 요소의 비율을 계산합니다.  
다음을 사용하여 새 줄에 각 분수의 소수 값을 인쇄하십시오.  
`6` 소수점 이하 자리.

<br />

## 참고

이 문제는 정밀도 문제를 야기합니다.  
 테스트 케이스는 소수점 이하 6자리까지 조정되지만 절대 오차가 최대 ` 10^-6`허용됩니다.

{" "}

<br />
## 예시 `arr = [1, 1, 0, -1, -1]`의 경우 `n = 5` 요소, 두 개의 양수, 두 개의 음수
및 하나의 0. 그들의 비율은 `2/5 = 0.400000`, `2/5 0.400000`그리고 `1/5 = 0.200000`.
결과는 다음과 같이 반환됩니다.

```
0.400000
0.400000
0.200000
```

<br />

## 기능 설명

아래 편집기에서 plusMinus 함수를 완성 하세요.

plusMinus에는 다음 매개변수가 있습니다.

- int arr[n] : 정수 배열

<br />

## Return

배열에서 양수, 음수 및 0 값의 비율을 인쇄합니다.  
각 값은 다음과 같이 별도의 줄에 인쇄되어야 합니다.  
소수점 이하 자릿수 `6` 개.  
함수는 값을 반환하지 않아야 합니다.

<br />

## 입력 형식

첫 번째 줄은 정수를 포함하고, `n`, 배열의 크기.
두 번째 줄에는 `n`설명하는 공백으로 구분된 정수 arr[n].

<br />

## 제약

`0 < n ≤ 100`
`-100 ≤ arr[i] ≤ 100`

<br />

## 출력 형식

다음을 인쇄 하십시오  
`3`라인, 각각 `6` 소수:

1. 양수 값의 비율
2. 음수 값의 비율
3. 0의 비율

<br />

## 샘플 입력

```
STDIN           Function
-----           --------
6               arr[] size n = 6
-4 3 -9 0 4 1   arr = [-4, 3, -9, 0, 4, 1]
```

<br />

## 샘플 출력

```
0.500000
0.333333
0.166667
```

<br />

## 설명

양수가 `3`, 음수가 `2` 0은 `1` 개 입니다.
발생 비율은 양수가 `3/6 = 0.500000`. 음수가 `2/6 = 0.333333`, 0이 `1/6 = 0.166667`.

<br />

## solution

```JS
  const handleRatios = (v, arr) => (v / arr.length).toFixed(6);

  const plusMinus = (arr) => {
    let positiveCount = 0;
    let negativeCount = 0;
    let zeroCount = 0;
    arr.forEach((v) => {
      if (v > 0) positiveCount += 1;
      if (v < 0) negativeCount += 1;
      if (v === 0) zeroCount += 1;
    });
    return (
      console.log(handleRatios(positiveCount, arr)),
      console.log(handleRatios(negativeCount, arr)),
      console.log(handleRatios(zeroCount, arr))
    );
  };
```
