# Divisible Sum Pairs

🔗 [problem](https://www.hackerrank.com/challenges/divisible-sum-pairs/problem?isFullScreen=true)

정수 배열과 양의 정수가 주어지면 `k`, 수를 결정 `(i, j)`쌍 어디에 `i < j`그리고 `ar[i]`+ `ar[j]`로 나눌 수 있다 `k`.

## 예시

`ar = [1, 2, 3, 4, 5, 6]`
`k = 5`

세 쌍이 기준을 충족합니다. `[1, 4]`, `[2, 3]`, 그리고 `[4, 6]`.

<br />

## 기능 설명

아래 에디터에서 divisibleSumPairs 함수를 완성 하세요.

divisibleSumPairs에는 다음 매개변수가 있습니다.

- int n: 배열의 길이 `ar`
- int ar[n]: 정수 배열
- int k: 정수 제수

<br />

## Returns

- int: 쌍의 수

<br />

## 입력 형식

첫 번째 줄에는 `2`공백으로 구분된 정수, `n`그리고 `k`.
두 번째 줄에는 `n`공백으로 구분된 정수, 각 값 `arr[i]`.

<br />

## 제약

- `2 ≤ n ≤ 100`
- `1 ≤ k ≤ 100`
- `1 ≤ ar[i] ≤ 100`

<br />

 5

<br />

## 설명 0

여기 `5` 유효한 쌍 `k = 3`:

- `(0, 2) -> ar[0] + ar[2] = 1 + 2 = 3`
- `(0, 5) -> ar[0] + ar[2] = 1 + 2 = 3`
- `(1, 3) -> ar[1] + ar[3] = 3 + 6 = 9`
- `(2, 4) -> ar[2] + ar[4] = 2 + 2 = 3`
- `(4, 5) -> ar[4] + ar[5] = 1 + 2 = 3`

<br />

## solution

```JS
const divisibleSumPairs = (n, k, ar) => {
    let count = 0;
    ar.forEach((x, i) => {
      while (i <= n) {
        i++;
        if ((x + ar[i]) % k === 0) count++;
      }
    });
    return count;
};
```
