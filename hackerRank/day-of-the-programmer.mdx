# Day of the Programmer

🔗 [problem](https://www.hackerrank.com/challenges/day-of-the-programmer/problem?isFullScreen=true)

마리는 타임머신을 발명했고 1700년부터 2700년까지의 범위를 포함하는 1년 동안 프로그래머의 날(연중 256일)에 러시아를 방문하기 위해 시간 여행을 통해 타임머신을 테스트하고 싶어합니다.

1700년부터 1917년까지 러시아의 공식 달력은 율리우스력이었다; 1919년부터 그들은 그레고리력 시스템을 사용했습니다. 율리우스력에서 그레고리력으로의 전환은 1918년 1월 31일 다음 날이 2월 14일 때 일어났습니다. 이것은 1918년 2월 14일이 러시아에서 32번째 날이었다는 것을 의미합니다.

두 달력 체계에서, 2월은 윤년 동안 29일, 그리고 다른 모든 해 동안 28일이 있는 유일한 달입니다. 율리우스력에서 윤년은 4로 나눌 수 있으며, 그레고리력에서 윤년은 다음 중 하나입니다.

- 400으로 나눌 수 있습니다.
- 4로 나눌 수 있고 100으로 나눌 수 없습니다.

1년이 주어지면, 그 해의 공식 러시아 달력에 따라 그 해의 256번째 날을 찾으세요. 그런 다음 dd.mm 형식으로 인쇄하십시오.yyyy, 여기서 dd는 두 자리 날짜, mm는 두 자리 월, yyyy는 `y`입니다.

예를 들어, 주어진 `y` = 1984. 1984는 4로 나눌 수 있으므로 윤년입니다. 1918년 이후의 윤년 256일은 9월 12일이므로 정답은 `1984.09.12`입니다.

## solution

```JS
const dayOfProgrammer = (year) => {
    let dayOfTheProgrammer = "";
    if (year === 1918) dayOfTheProgrammer = "26";
    if (year > 1918) {
      (year % 4 === 0 && year % 100 !== 0) || year % 400 === 0
        ? (dayOfTheProgrammer = "12")
        : (dayOfTheProgrammer = "13");
    }
    if (year < 1918) {
      year % 4 === 0
        ? (dayOfTheProgrammer = "12")
        : (dayOfTheProgrammer = "13");
    }
    return `${dayOfTheProgrammer}.09.${year}`;
};
```

def dayOfProgrammer(year):

const dayOfProgrammer = (year) => {
    if (year === 1918) return `26.09.${year}`;
    if (
      (year > 1918 && year % 4 === 0 && year % 100 !== 0) ||
      year % 400 === 0 ||
      (year < 1918 && year % 4 === 0)
    )
      return `12.09.${year}`;

};
