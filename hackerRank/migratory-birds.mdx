# Migratory Birds

🔗 [problem](https://www.hackerrank.com/challenges/migratory-birds/problem?isFullScreen=true)

모든 요소가 새 유형 ID를 나타내는 새 목격 배열이 주어지면 가장 자주 목격되는 유형의 ID를 결정하십시오. 1개 이상의 유형이 최대 금액만큼 발견되면 가장 작은 ID를 반환합니다.

## 예시

`arr = [1, 1, 2, 2, 3]`

종류는 각각 2가지 `1` 그리고 `2`, 그리고 유형의 한 목격 `3`. 두 번 본 두 가지 유형 중 더 낮은 유형을 선택하십시오. `1`.

<br />

## 기능 설명

아래 편집기에서 migratoryBirds 기능을 완료하십시오 .

migratoryBirds에는 다음 매개변수가 있습니다.

- int arr[n] : 목격된 새의 종류

<br />

## Returns

- int: 가장 자주 목격되는 새의 가장 낮은 유형 ID

<br />

## 입력 형식

첫 번째 줄은 정수를 포함하고, `n`, 의 크기 `arr`.
두 번째 줄은 다음을 설명합니다. `arr`~처럼 `n`공백으로 구분된 정수로, 각각 목격된 새의 유형 번호입니다.

<br />

## 제약

- `5 ≤ n ≤ 2 X 10^5`
- 각 유형은 다음과 같습니다. `1`, `2`, `3`, `4`, 또는 5

<br />

## 샘플 입력 0

```
6
1 4 4 4 5 3
```

<br />

## 샘플 출력 0

```
4
```

<br />

## 설명 0

다양한 유형의 새가 다음 ​​빈도로 발생합니다.

- 유형 `1`: `1`새
- 유형 `2`: `0`새
- 유형 `3`: `1`새
- 유형 `4`: `3`새
- 유형 `5`: `1`새

가장 높은 빈도에서 발생하는 유형 번호는 유형입니다. `4`, 그래서 우리는 인쇄 `4`우리의 대답으로.

<br />

## 샘플 입력 1

```
11
1 2 3 4 5 4 3 2 1 3 4
```

<br />

## 샘플 출력 1

```
 3
```

<br />

## 설명 1

다양한 유형의 새가 다음 ​​빈도로 발생합니다.

- 유형 `1`: `2`새
- 유형 `2`: `2`새
- 유형 `3`: `3`새
- 유형 `4`: `3`새
- 유형 `5`: `1`새

두 가지 유형의 빈도는 `3`, 그리고 그 중 낮은 유형은 `3`.

<br />

## solution

```JS
const migratoryBirds = (arr) => {
    const typeOne = arr.filter((x) => x === 1).length;
    const typeTwo = arr.filter((x) => x === 2).length;
    const typeThree = arr.filter((x) => x === 3).length;
    const typeFour = arr.filter((x) => x === 4).length;
    const typeFive = arr.filter((x) => x === 5).length;
    const typeArr = [typeOne, typeTwo, typeThree, typeFour, typeFive];
    return typeArr.findIndex((x) => x === Math.max(...typeArr)) + 1;
};
```
