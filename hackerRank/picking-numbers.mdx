# Picking Numbers

🔗 [Problem](https://www.hackerrank.com/challenges/picking-numbers/problem?isFullScreen=true)

정수 배열이 주어졌을 때 두 원소 사이의 절대적인 차이가 1 이하인 가장 긴부분 배열을 구하시오.

<br />

## 예제

```
a = [1, 1, 2, 2, 4, 4, 5, 5, 5]
```

기준을 충족하는 두 개의 하위 배열 [1, 1, 2, 2]와 [4, 4, 5, 5, 5] 가 있습니다. 최대 길이 하위 배열에는 5개의 요소가 있습니다.

<br />

## 기능 설명

아래 에디터에서 pickingNumbers 함수를 완성하세요.
pickingNumbers는 다음의 매게변수를 가지고 있습니다.

- int a[n] : 정수의 배열

<br />

## Return

- int: 기준을 충족하는 가장 긴 하위 배열의 길이

<br />

## 입력 형식

첫 번째 줄에는 배열 a의 크기인 단일 정수 n이 포함됩니다.
두 번째 줄에는 공백으로 구분된 n개의 정수가 포함되어 있으며 각 정수는 a[i]입니다.

<br />

## 제약

`2 ≤ n ≤ 100`  
`0 ≤ a[i] ≤ 100 `
`answer ≥ 2`

<br />

## 샘플 Input 0

```
6
4 6 5 3 3 1
```

<br />

## 샘플 Output 0

```
3
```

<br />

## 설명 0

배열에서 {4, 3, 3}의 정수 다중 집합을 선택합니다.  
다중 집합의 각 쌍은 절대적인 차이 ≤ 1(즉, | 4 - 3 | = 1, 및 | 3 - 3 | = 0)을 가지고 있으므로 선택된 정수의 수인 3을 답으로 인쇄합니다.

<br />

## 샘플 Input 1

```
6
1 2 2 3 1 2
```

<br />

## 샘플 Output 1

```
5
```

<br />

## 설명 1

배열에서 정수의 다중 집합 {1, 2, 2, 1, 2}을 선택합니다.  
다중 집합의 각 쌍은 절대 차이 ≤ 1(즉, | 1 - 2 | = 1, | 1 - 1 | = 0, | 2 - 2 | = 0)을 가지고 있으므로 선택된 정수의 수 5를 답으로 인쇄합니다.

<br />

## solution

```JS
const pickingNumbers = (a) =>
    Math.max(
      ...[...new Set(a)].map(
        (x) => a.filter((y) => y === x || y === x + 1).length
      )
);
```
