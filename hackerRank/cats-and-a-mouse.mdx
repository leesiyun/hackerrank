# Cats and a Mouse

🔗 [Problem](https://www.hackerrank.com/challenges/cats-and-a-mouse/problem?isFullScreen=true)

고양이 두 마리와 쥐 한 마리가 한 줄에 다양한 위치에 있습니다.  
당신은 그 것들의 출발 위치가 주어집니다.  
당신의 임무는 쥐가 움직이지 않고 고양이들이 같은 속도로 이동한다고 가정했을 때 어떤 고양이가 먼저 마우스에 도달할지 결정하는 것입니다.  
만약 고양이들이 같은 시간에 도착한다면, 쥐는 움직일 수 있게 되고 고양이들이 싸우는 시간 동안 탈출하게 될 것입니다.

당신에게는 `A` 그리고 `B` 고양이와 `C`라는 쥐에 대한 각각의 위치는 `x` ,`y` , 그리고 `z` 의 형태로 `q`라는 쿼리가 주어집니다.
각 쿼리에 대한 적절한 대답을 반환하는 함수를 완성하십시오.

- `A`고양이가 먼저 쥐를 잡으면 Cat A를 출력합니다.
- `B`고양이가 먼저 쥐를 잡으면 Cat B를 출력합니다.
- 두 고양이가 동시에 쥐에게 도달하면, 두 고양이가 싸우는 동안 쥐가 탈출하는데 이 때 Mouse C를 출력합니다.

<br />

## 예제

```
x = 2
y = 5
z = 4
```

고양이들은 2(Cat A)와 5(Cat B) 위치에 있고, 쥐는 5(Cat B) 쪽에 더 가깝게 위치게 있습니다.
그렇기 때문에 Cat B는를 반환합니다.

<br />

## 기능 설명

아래의 편집기에서 catAndMouse 기능을 완료하세요
catAndMouse의 매개 변수는 다음과 같습니다.

- int x : 고양이의 위치
- int y : 고양이의 위치
- int z : 쥐의 위치

<br />

## Return

- string: 'Cat A', 'Cat B', 또는 'Mouse C'

<br />

## 입력 형식

첫 번째 줄에는 쿼리 수를 나타내는 정수 `q`가 하나가 포함되어 있습니다.
`q`의 각 행에는 x(고양이 A의 위치), y(고양이 B의 위치) 및 z(쥐 C의 위치)의 각 값을 설명하는 세 개의 공백으로 구분된 정수가 포함됩니다.

<br />

`1 ≤ x, y, z ≤ 100 `

## 설명 0

쿼리 0 : 고양이와 마우스의 위치는 다음과 같습니다.
![](https://s3.amazonaws.com/hr-challenge-images/0/1480434477-7418fccf34-cat.png)

`B`고양이가 먼저 쥐를 잡을 것이기 떄문에 우리는 `Cat B`를 출력합니다.

쿼리 1: 이 쿼리에서는 `A` 그리거 `B` 고양이와 쥐 `C`를 정확하게 연결되어 있습니다.

![](https://s3.amazonaws.com/hr-challenge-images/0/1480434557-601bef86ba-cat1.png)

고양이들은 동일한 시간에 도착하기 때문에 쥐는 탈출합니다. 그러므로 `Mouse C`를 출력합니다.

<br />

## solution

```JS
const catAndMouse = (x, y, z) => {
    const catA = Math.abs(z - y);
    const catB = Math.abs(z - x);
    if (catA > catB) return 'Cat A';
    if (catA < catB) return 'Cat B';
    if (catA === catB) return 'Mouse C';
};
```
