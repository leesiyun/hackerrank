# A Very Big Sum

🔗 [Problem](https://www.hackerrank.com/challenges/a-very-big-sum/problem?isFullScreen=true)

이 과제에서는 일부 정수가 상당히 클 수 있다는 점을 염두에 두고 배열에 있는 요소의 합을 계산하고 인쇄해야 합니다.

<br />

## 기능 설명

아래 편집기에서 매우 큰 합계 기능을 완료하십시오 . 모든 배열 요소의 합계를 반환해야 합니다.

aVeryBigSum에는 다음 매개변수가 있습니다.

- int ar[n] : 정수 배열.

<br />

## Return

- long : 모든 배열 요소의 합

<br />

## 입력 형식

입력의 첫 번째 줄은 정수로 구성됩니다. `n`.  
다음 줄에는 다음이 포함됩니다. `n` 배열에 포함된 공백으로 구분된 정수입니다.

<br />

## 출력 형식

배열에 있는 요소의 정수 합계를 반환합니다.

<br />

## 제약

`1 ≤ n ≤ 100`  
`1 ≤ ar[i] ≤ 100 `

<br />

## 샘플 입력

```
5
1000000001 1000000002 1000000003 1000000004 1000000005
```

<br />

## 샘플 출력

```
5000000015
```

<br />

## 설명

32비트 정수의 범위는 `(-2^31) to (2^31 -1) or [-2147483648, 2147483637]`.  
여러 정수 값을 추가하면 결과 합계가 위의 범위를 초과할 수 있습니다. 이러한 합계를 저장하려면 long int C/C++/Java를 사용해야 할 수도 있습니다.

<br />

## solution

```JS
const aVeryBigSum = (ar) => ar.reduce((acc, cur) => acc + cur);
```
