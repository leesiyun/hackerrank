# Subarray Division

🔗 [problem](https://www.hackerrank.com/challenges/the-birthday-bar/problem?isFullScreen=true)

릴리와 론은 초콜릿 바를 공유하고 싶어합니다. 각 정사각형에는 정수가 있습니다.

Lily는 다음과 같이 선택된 막대의 연속적인 부분을 공유하기로 결정합니다.

- 세그먼트의 길이는 Ron의 생일인 달과 일치하고,
- 정사각형에 있는 정수의 합은 그의 생일인 일과 같습니다.

그녀가 초콜릿을 나눌 수 있는 방법의 수를 결정하십시오.

## 예시

`s = [2, 2, 1, 3, 2]`
`d = 4`
`m = 2`

릴리는 론의 생일을 합산하는 세그먼트를 찾고자 합니다. `d = 4`그의 출생 월과 같은 길이로, `m =2`. 이 경우 그녀의 기준을 충족하는 두 개의 세그먼트가 있습니다. `[2, 2]`그리고 `[1, 3]`.

## 입력 형식

첫 번째 줄에는 정수가 포함되어 있습니다. `n`, 초콜릿 바의 사각형 수.  
두 번째 줄에는 `n`공백으로 구분된 정수 `s[i]`, 초콜릿 사각형의 숫자는 `0 ≤ i < n`.  
세 번째 줄에는 공백으로 구분된 두 개의 정수가 포함되어 있습니다. `d`그리고 `m`, Ron의 생년월일과 생년월일.

## 제약

- `1 ≤ n ≤ 100`
- `1 ≤ s[i] ≤ 5, where(0 ≤ i < n)`
- `1 ≤ d ≤ 31`
- `1 ≤ m ≤ 12`

<br />

## solution

```JS
const birthday = (s, d, m) => {
    let count = 0;
    s.forEach((el, i) => {
      if (s.slice(i, i + m).reduce((acc, cur) => acc + cur) === d) count++;
    });
    return count;
};
```
