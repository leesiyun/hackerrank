# Between Two Sets

🔗 [problem](https://www.hackerrank.com/challenges/between-two-sets/problem?isFullScreen=true)

두 개의 정수 배열이 있습니다. 모든 정수들이 다음 두 조건에 만족하는지 확인하세요.

1. 첫 번째 배열의 요소들은 고려되는 정수의 모든 인수들(factors)
2. 고려되는 정수는 두 번째 배열의 모든 요소들의 인수(factor)

이 숫자들은 두 배열 사이에 있을 때 그러한 숫자가 몇 개인지 반환합니다.

## 예시

```
a = [2, 6]
b = [24, 36]
```

위 배열에서는 `6` 그리고 `12` 두 개의 숫자가 있습니다.  
첫 번째는 `6 % 2 = 0`, `6 % 6 = 0`, `24 % 6 = 0`그리고 `36 % 6 = 0` 이고,  
두 번째는 `12 % 2 = 0`, `12 % 6 = 0`그리고 `24 % 12 = 0`, `36 % 12 = 0` 이므로 `2`를 반환합니다.

<br />

## 기능 설명

getTotalX 함수를 완성 하십시오. 집합 사이의 정수 수를 반환해야 합니다.

getTotalX에는 다음 매개변수가 있습니다.

- int a[n] : 정수 배열
- int b[m] : 정수 배열

<br />

## Returns

- int: 집합 사이에 있는 정수의 수

<br />

## 입력 형식

첫 번째 줄에는 공백으로 구분된 두 개의 정수가 포함되어 있습니다.그리고, 배열의 요소 수 `a` 그리고 `b`.  
두 번째 줄에는 `n` 고유한 공백으로 구분된 정수 `a[i]` 어디 `0 ≤ i < n`.  
세 번째 줄에는 다음이 포함됩니다. `m`고유한 공백으로 구분된 정수 `b[j]`어디 `0 ≤ j < m`.

<br />

## 제약

- `1 ≤ n, m ≤ 10`
- `1 ≤ a[i] ≤ 100`
- `1 ≤ b[j] ≤ 100`

<br />

## 샘플 입력 0

```
2 3
2 4
16 32 96
```

<br />

## 샘플 출력 0

```
3
```

<br />

## 설명 0

2와 4는 4, 8, 12, 16
으로 균등하게 나눕니다. 4, 8 및 16은 16, 32, 96으로 균등하게 나눕니다.

4, 8, 16은 a의 각 요소가 요소이고 각각이 b의 모든 요소의 요소인 유일한 세 숫자입니다.

## 다른 사람의 풀이 살펴보기

```JS
function getTotalX(a, b) {
    let count = 0
    let na = Math.max(...a)
    let mi = Math.min(...b)
    for(let i=na; i<=mi ;i++){ // Loop between Array[a](MAX) and Array[b](MIN)
        if((a.every(e => i % e === 0)) && b.every(e => e % i === 0)){
// Check if the i(INTEGER) is Factor of Array[a](ELEMENTS) & Array[b](ELEMENTS) are Factor of the i(INTEGER)
            count++ // add +1 for each found integer
        }
    }
     return count
}
```

<br />

## solution

```JS
const getTotalX = (a, b) => {
    let count = 0;
    for (let i = a.slice(-1)[0]; i <= b[0]; i++) {
      if (a.every((e) => i % e === 0) && b.every((e) => e % i === 0)) count++;
    }
    return count;
};
```
