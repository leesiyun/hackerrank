# The Hurdle Race

🔗 [Problem](https://www.hackerrank.com/challenges/the-hurdle-race/problem?isFullScreen=true)

비디오 플레이어는 캐릭터가 허들 경주에서 경쟁하는 게임을 합니다.  
허들은 높이가 다양하고, 등장인물들은 점프할 수 있는 최대 높이를 가지고 있습니다.  
그들이 복용할 수 있는 마법의 물약이 있는데, 이 물약을 복용할 때마다 최대 점프 높이가 `1`단위씩 증가합니다.  
캐릭터가 모든 장애물을 뛰어넘기 위해서는 몇 도스의 물약이 필요한가요.  
캐릭터가 이미 모든 장애물을 제거할 수 있는 경우 `0`을 반환합니다.

<br />

## 예제

```
height = [1, 2, 3, 3, 2]
k = [1]
```

캐릭터는 처음에 `1`단위 높이로 점프할 수 있고  
모든 장애물을 뛰어넘기 위해서는 `3 - 1 = 2`의 물약을 사용해야 합니다.

<br />

## 기능 설명

아래 에디터에서 hurdleRace 함수를 작성하세요.

hurdleRace에는 다음 매개 변수가 있습니다:

- int k: 캐릭터가 점프할 수 있는 높이입니다
- int heigh[n]: 각 장애물의 높이입니다

<br />

## Return

- int: 필요한 최소 물약 수, 항상 '0' 이상입니다

<br />

## 입력 형식

첫 번째 줄에는 띄어쓰기로 분리된 두 개의 정수 `n`과 `k`,  
즉 허들의 수와 캐릭터가 자연스럽게 점프할 수 있는 최대 높이가 포함되어 있습니다.
두 번째 줄에는 띄어쓰기로 분리된 정수 `n`개의 `height[i]`가 포함되어 있으며, 여기서 `0 ≤ i ≤ n`입니다.

<br />

## 제약

`1 ≤ n, k ≤ 100`  
`1 ≤ height[i] ≤ 100 `

<br />

## 샘플 Input 0

```
5 4
1 6 3 5 2
```

<br />

## 샘플 Output 0

```
2
```

<br />

## 설명 0

댄의 캐릭터는 최대 `k = 4` 단위까지 점프할 수 있지만, 가장 높은 장애물의 높이는 `h[1] = 6`입니다:  
![](https://s3.amazonaws.com/hr-challenge-images/0/1485458269-d39e09fb78-hurdle.png)  
모든 허들을 뛰어넘기 위해서는 댄이 6 - 4 = 2를 마셔야 합니다.

<br />

## 샘플 Input 1

```
5 7
2 5 4 5 2
```

<br />

## 샘플 Output 1

```
0
```

<br />

## 설명 1

댄의 캐릭터는 최대 `k = 7` 단위까지 점프할 수 있으며, 이는 모든 장애물을 뛰어넘기에 충분합니다:  
![](https://s3.amazonaws.com/hr-challenge-images/0/1485458562-e680371e5a-hurdle1.png)  
댄은 이미 모든 허들을 뛰어넘을 수 있기 때문에 물약을 마시지 않아도 됩니다.

<br />

## solution

```JS
const hurdleRace = (k, height) =>
    Math.max.apply(null, height) - k > 0 ? Math.max.apply(null, height) - k : 0;
```
