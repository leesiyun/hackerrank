# Mini-Max Sum

🔗 [Problem](https://www.hackerrank.com/challenges/mini-max-sum/problem?isFullScreen=true)

5개의 양의 정수가 주어졌을 때 5개의 정수 중 정확히 4개를 더하여 계산할 수 있는 최소값과 최대값을 찾으십시오. 그런 다음 각각의 최소값과 최대값을 공백으로 구분된 두 개의 긴 정수로 된 한 줄로 인쇄합니다.

## 예시

`arr = [1, 3, 5, 7, 9]`

최소 합계는 `1 + 3 + 5 + 7 = 16`최대 합계는 `3 + 5 + 7 + 9 = 24`. 함수가 인쇄합니다.

```
16 24
```

<br />

## 기능 설명

아래 에디터에서 miniMaxSum 함수를 완성 하세요.

miniMaxSum에는 다음 매개변수가 있습니다.

- arr : 배열 `5` 정수

<br />

## Print

한 줄에 두 개의 공백으로 구분된 정수 인쇄: 최소 합 과 최대 합 `4`의 `5`집단.

<br />

## 입력 형식

공백으로 구분된 5개의 정수로 구성된 한 줄입니다.

<br />

## 제약

`1 ≤ arr[i] ≤ 10^9`

<br />

## 출력 형식

다섯 개의 정수 중 정확히 네 개 를 합하여 계산할 수 있는 각각의 최소값과 최대값을 나타내는 두 개의 공백으로 구분된 긴 정수를 인쇄하십시오 . (출력은 32비트 정수보다 클 수 있습니다.)

<br />

## 샘플 입력

```
1 2 3 4 5
```

<br />

## 샘플 출력

```
10 14
```

<br />

## 설명

숫자는 `1`, `2`, `3`, `4`, 그리고 `5`. 5개의 정수 중 4개를 사용하여 다음 합계를 계산합니다.

제외한 모든 것을 합산 `1`, 합계는 `2 + 3 + 4 + 5 = 4`.  
제외한 모든 것을 합산 `2`, 합계는 `1 + 3 + 4 + 5 = 13`.  
제외한 모든 것을 합산 `3`, 합계는 `1 + 2 + 4 + 5 = 12`.  
제외한 모든 것을 합산 `4`, 합계는 `1 + 2 + 3 + 5 = 11`.  
제외한 모든 것을 합산 `5`, 합계는 `1 + 2 + 3 + 4 = 10`.

<br />

## 흰트

정수 오버플로에 주의하십시오! 64비트 정수를 사용합니다.

<br />

## solution

```JS
const miniMaxSum = (arr) => {
    arr.sort();
    let min = arr.slice(0, 4).reduce((acc, cur) => acc + cur);
    let max = arr.slice(1, 5).reduce((acc, cur) => acc + cur);
    return console.log(`${min} ${max}`);
};
```

## 다른 사람의 풀이 살펴보기

```JS
function miniMaxSum(arr) {
    let sum = arr.reduce((a, b) => a + b);
    let maxVal = Math.max(...arr);
    let minVal = Math.min(...arr);
    console.log((sum - maxVal) + ' ' + (sum - minVal));
}
```
