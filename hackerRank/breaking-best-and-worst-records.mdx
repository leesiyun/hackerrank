# Breaking the Records

🔗 [problem](https://www.hackerrank.com/challenges/breaking-best-and-worst-records/problem?isFullScreen=true)

Maria는 대학 농구를 하고 프로가 되고 싶어합니다. 매 시즌 그녀는 자신의 플레이 기록을 유지합니다. 그녀는 한 경기에서 가장 많은 득점 과 최소 득점 에 대한 자신의 시즌 기록을 깨뜨린 횟수를 표로 작성합니다 . 첫 번째 게임에서 득점한 득점은 시즌 기록을 세우며 그녀는 거기부터 계산을 시작합니다.

## 예시

`scores = [12, 24, 10, 24]`

점수는 플레이한 게임과 같은 순서입니다. 그녀는 결과를 다음과 같이 표로 작성합니다.

```
                                     Count
    Game  Score  Minimum  Maximum   Min Max
     0      12     12       12       0   0
     1      24     12       24       0   1
     2      10     10       24       1   1
     3      24     10       24       1   1
```

시즌 점수가 주어지면 Maria가 시즌 동안 득점 한 최고 및 최저 점수에 대한 기록을 몇 번이나 깨는지 확인합니다.

<br />

## 기능 설명

아래 편집기에서 breakRecords 기능을 완료 하세요.

breakRecords에는 다음 매개변수가 있습니다.

- int Score[n]: 게임당 득점한 점수

<br />

## Returns

- int[2]: 기록을 깨뜨린 횟수가 포함된 배열입니다. 색인 `0` 대부분의 점수 기록 을 깨기 위한 것이며 색인 `1` 최소 점수 기록 을 깨기 위한 것 입니다.

<br />

## 입력 형식

첫 번째 줄에는 정수가 포함되어 있습니다.`n`, 게임 수.
두 번째 줄에는 `n` 각 값을 설명하는 공백으로 구분된 정수 `score0, score1, ... , scoren-1`.

<br />

## 제약

- `1 ≤ n ≤ 1000`
- `0 ≤ scores[i] ≤ 10^8`

## 설명 1

아래 도표는 마리아가 시즌 동안 최고 기록과 최악 기록을 깨뜨린 횟수를 보여줍니다.

![](https://s3.amazonaws.com/hr-assets/0/1487360375-aee4388234-breakingbest5.png)

그녀는 자신의 최고 기록을 4번이나 깼습니다(경기 후, `1`, `2`, `3` 그리고) 그리고 그녀의 최악의 기록은 0번입니다(시즌 중 점수가 첫 게임에서 얻은 점수보다 낮지 않음). 그래서 우리 4 0는 우리의 답으로 인쇄합니다.

<br />

## solution

```JS
const breakingRecords = (scores) => {
    let max = scores[0];
    let maxCount = 0;
    let min = scores[0];
    let minCount = 0;
    scores.map((score) => {
      if (score > max) {
        maxCount++;
        max = score;
      }
      if (score < min) {
        minCount++;
        min = score;
      }
    });
    return [maxCount, minCount];
};
```
