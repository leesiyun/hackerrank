# Compare the Triplets

🔗 [problem](https://www.hackerrank.com/challenges/compare-the-triplets/problem?isFullScreen=true)

Alice와 Bob은 각각 HackerRank에 대해 하나의 문제를 만들었습니다.  
검토자는 문제 명확성 , 독창성 및 난이도 의 세 가지 범주에 대해 1 에서 100 까지의 점수를 부여하여 두 가지 과제를 평가합니다 .

Alice의 챌린지에 대한 등급은 3중항 a = (a[0], a[1], a[2]) 이고 Bob의 챌린지에 대한 등급은 3중항 b = (b[0], b[1], b[2]) 입니다.

작업은 a[0] 과 b[0] , a[1] 과 b[1] , a[2] 와 b[2] 를 비교하여 비교점 을 찾는 것 입니다.

- a [i] > b[i] 이면 Alice는 1 점을 받습니다.
- a [i] < b[i] 이면 Bob은 1 점을 받습니다.
- a [i] = b[i] 이면 어느 누구도 점수를 받지 못합니다.
  비교 포인트는 개인이 획득한 총 포인트입니다.

주어진 a 와 b 각각의 비교점을 결정하십시오.

<br />

## 예시

a = [1, 2, 3]  
b = [3, 2, 1]

- **0** 요소의 경우 a[0] 으로 인해 Bob에게 점수가 부여됩니다.
- 동일한 요소 a[1] 및 b[1] 의 경우 포인트가 적립되지 않습니다.
- 마지막으로 요소 2 의 경우 a[2] > b[2] 이므로 Alice는 포인트를 받습니다.

반환 배열은 [1, 1] 이며 Alice의 점수가 먼저이고 Bob의 점수가 두 번째입니다.

<br />

## 기능 설명

아래 편집기에서 compareTriplets 기능을 완료하십시오 .

compareTriplets에는 다음 매개변수가 있습니다.

- int a[3] : 앨리스의 도전 등급
- int b[3] : 밥의 도전 등급

<br />

## Return

- int[2] : Alice의 점수가 첫 번째 위치에 있고 Bob의 점수가 두 번째 위치에 있습니다.

<br />

## 입력 형식

첫 번째 줄에는 3개의 공백으로 구분된 정수 a[0] , a[1] , a[2] 가 포함 되어 있으며, 각각의 값은 a 입니다.  
두 번째 줄에는 공백으로 구분된 3 개의 정수 b[0] , b[1] , b[2] 가 포함 되어 있으며, 이는 삼중항 b 의 각 값입니다 .

<br />

## 제약

- 1 ≤ a[i] ≤ 100
- 1 ≤ b[i] ≤ 100

<br />

## 샘플 입력 0

```
5 6 7
3 6 10
```

<br />

## 샘플 출력 0

```
1 1
```

<br />

## 설명 0

이 예에서:

- `a = (a[0], a[1], a[2]) = (5, 6, 7)`
- `b = (b[0], b[1], b[2]) = (3, 6, 10)`

이제 각 개별 점수를 비교하겠습니다.

- `a[0] > b[0]`, 그래서 앨리스는 1 포인트를 얻습니다.
- `a[1] = b[1]`, 그래서 아무도 포인트를 받지않습니다.
- `a[2] < b[2]`, 그래서 밥은 1 포인트를 얻습니다.  
  앨리스의 비교 점수는 `1`, 그리고 Bob의 비교 점수는 `1`. 따라서 우리는 배열을 반환합니다. `[1, 1]`.

<br />

## 샘플 입력 1

```
17 28 30
99 16 8
```

<br />

## 샘플 출력 1

```
2 1
```

<br />

## 설명 1

비교 `0th` 집단, `17 < 99` 그래서 Bob은 포인트를 받습니다.  
비교 `1st` 그리고 `2nd `집단, `28 > 16`그리고 `30 > 8`그래서 앨리스는 2점을 받습니다.  
반환 배열은 `[2, 1]`.

<br />

## solution

```JS
const compareTriplets = (a, b) => {
	let aPoint = a.filter((el, i) => el > b[i]).length;
	let bPoint = b.filter((el, i) => el > a[i]).length;
	return [aPoint, bPoint];
};

```
