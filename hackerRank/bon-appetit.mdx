# Bill Division

🔗 [problem](https://www.hackerrank.com/challenges/bon-appetit/problem?isFullScreen=true)

두 친구 Anna와 Brian은 저녁 ​​식사에서 청구서를 어떻게 나눌지 결정하고 있습니다. 각자 소비한 항목에 대해서만 비용을 지불합니다. Brian은 확인과 Anna의 몫을 계산합니다. 그의 계산이 정확한지 확인해야 합니다.

예를 들어, 청구서의 가격이 `bill = [2, 4, 6]`라고 가정합니다. 안나는 `k = bill[2]`의 가격이 `6`인 아이템을 먹는 것을 거부합니다. 브라이언이 계산한 계산이 정확하다면, 안나는 `(2 + 4)/2`를 지불할 것입니다. 만약 그가 `bill[2]`의 비용을 포함한다면, 그는 `(2 + 4 + 6)/2`를 계산할 것이다. 두 번째 경우, 그는 안나에게 `3`을 환불해야 합니다.

<br />

## 기능 설명

아래 편집기에서 bonAppetit 기능을 완료합니다. 청구서가 상당히 분할되어 있다면 Bon Aftit라고 인쇄되어야 한다. 그렇지 않으면, 브라이언이 안나에게 빚진 돈을 프린트해야 한다.

bonAppetit에는 다음 매개 변수가 있습니다.

- bill: 주문된 각 항목의 비용을 나타내는 정수 배열
- k: Anna가 먹지 않는 항목의 0 기반 인덱스를 나타내는 정수
- b: 안나가 그 법안에 기여한 돈의 양

<br />

## 입력 형식

첫 번째 줄에는 공백으로 구분된 두 개의 정수 `n`과 `k`가 포함되어 있으며, 주문된 항목의 수와 안나가 먹지 않은 항목의 `0` 기반 인덱스가 포함되어 있습니다.
두 번째 줄에는 `n`개의 공백으로 구분된 정수 `bill[i]`가 포함되어 있습니다. 여기서 `0 ≤ n < n`입니다.
세 번째 줄에는 브라이언이 안나에게 청구한 금액인 정수 `b`가 포함되어 있다.

<br />

## 제약

- `2 ≤ n ≤ 10^5`
- `0 ≤ k ≤ n`
- `0 ≤ bill[i] ≤ 10^4`
- `0 ≤ b ≤ sum i=0 to (n-i) bill[i]`
- Anna가 지불해야 할 돈의 양은 항상 정수입니다.

<br />

## 출력 형식

Brian이 Anna를 과대 청구하지 않은 경우, Bon Aftit를 새 줄에 인쇄하고, 그렇지 않은 경우 Brian이 Anna에게 환불해야 하는 차액(즉, b[지불한 값] - b[실제 값])을 인쇄합니다. 이 값은 항상 정수입니다.

<br />

## 샘플 입력 0

```
4 1
3 10 2 9
12
```

<br />

## 샘플 출력 0

```
5
```

<br />

## 설명 0

안나는 "bill[1] = 10" 항목을 먹지 않았지만 브라이언과 나머지 항목을 공유했습니다. 공유 항목의 총 비용은 "3 + 2 + 9 = 14"이고, 반으로 나누면 1인당 비용은 "b[실제 값] = 7"입니다. Brian은 청구서의 그녀의 부분에 대해 "b[지불한 값] = 12"를 청구했습니다. 우리는 Anna가 초과 청구된 금액인 "b[지불한값] - b[실제값] = 12 - 7 = 5"를 반환합니다.

<br />

```
3 10 2 9
7
```

## 샘플 출력 1

```
Bon Appetit
```

## 설명 1

안나는 `bill[1] = 10` 항목을 먹지 않았지만 브라이언과 나머지 항목을 공유했습니다. 공유 항목의 총 비용은 `3 + 2 + 9 = 14`이고, 반으로 나누면 1인당 비용은 `b[실제 값] = 7`입니다. `b[실제값] = b[지불한값] = 7`이기 때문에 Bon Appetit를 반환합니다..

<br />

## solution

```JS
const bonAppetit = (bill, k, b) =>
    (bill.reduce((acc, cur) => acc + cur) - bill[k]) / 2 === b
      ? console.log("Bon Appetit")
      : console.log(bill[k] / 2);
```

<br />

## 다른 사람의 풀이 살펴보기

```Python
n, k = map(int, input().strip().split(' '))
c = [int(x) for x in input().strip().split(' ')]

print('Bon Appetit' if (sum(c) - c[k]) // 2 == b else c[k] // 2)
```
